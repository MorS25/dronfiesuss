## Using from leaflet
To use from leaflet it's important get the workspace and layer name.
```
L.tileLayer.wms('http://localhost:8080/geoserver/Dronfies/wms?', {
          service: "WMS",
          version: "1.1.0",
          request: "GetMap",
          layers: "Dronfies:HYP_HR_SR_OB_DR",
          srs: "EPSG:4326",
        }
```
